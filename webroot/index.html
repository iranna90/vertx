Introduction

Irdeto Control is a core component of the Irdeto Rights family of products. It is a comprehensive solution to secure content delivery and media distribution management.

Irdeto Control provides three core functions for handling both downloadable and streaming content:

  • Content protection: protects content and enforces restrictions on how content can be consumed.
  • Access control: an additional layer of protection that enables operators to impose business policies about where, when, and how the content can be accessed
  • Integration into a broader ecosystem: a set of REST APIs that facilitates the integration of operator applications with Irdeto Control.

Irdeto Control simplifies the management of live and on demand OTT offerings. It supports two deployment models:

  • as a fully managed service in the cloud
  • as a licensed software application (on-premises)
